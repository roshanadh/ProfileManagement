# User Profile Management
An MVC application for user profile management

## Features
* User sign-in (with *HttpSession*)
* User registration
* List all users
* Update user profile
* Delete user profile
* Search for users

## Technology stack
* Java Servlet
* JavaServer Pages (JSP)
* JavaServer Pages Standard Tag Library (JSTL)
* Java Database Connectivity (JDBC)
* MySQL
* Bootstrap
